# 操作方法
コントローラー(Xinput)対応<br>
Lスティック          : 移動<br>
Rスティック          : カメラ操作<br>
Rスティック押し込み  : ロックオン切り替え<br>
X                    : 弱攻撃<br>
Y                    : 強攻撃<br>
B　		     : 回避<br>
A　		     : 決定<br>
L１＋（A.X.Y.B）     : スキル3種<br>
R１　		     : ガード<br>
L１＋R１　	     : 覚醒<br>
