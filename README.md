visualStudio 2022

C++

# 操作方法
コントローラー(Xinput)対応\n
Lスティック　　　　　: 移動 
Rスティック　　　　　: カメラ操作\r\n

Rスティック押し込み　: ロックオン切り替え

X　		　　 : 弱攻撃

Y　		     : 強攻撃

B　		     : 回避

A　		     : 決定

L１＋（A.X.Y.B）     : スキル3種

R１　		     : ガード

L１＋R１　	     : 覚醒
